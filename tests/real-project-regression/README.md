# 真实项目创作回归样本

本目录保存三个代表章节的 Prompt 与正文样本，覆盖资源谈判、科研试验和终局战争。

样本用于人工阅读以下创作关系：

- Prompt 是否把章纲转成人物目标、阻力、策略、反制、选择和后果。
- writer 是否把关键场景写成实际发生的过程。
- 技术、战斗和谈判是否通过人物行动推进，而不是结果播报。
- 人物知识、资源、能力、时间和章末状态是否连续。
- Prompt 中的文风与题材材料是否落实到正文表达。

## 样本文件

```text
prompts/vol-2-ch-23.md
drafts/vol-2-ch-23.md

prompts/vol-2-ch-41.md
drafts/vol-2-ch-41.md

prompts/vol-2-ch-81.md
drafts/vol-2-ch-81.md
```

人工回归按章节顺序阅读 Prompt 和正文，根据人物行动、因果、信息、关系、节奏和收束形成判断。不运行关键词、词频、覆盖率、AI 味或文学评分脚本。
